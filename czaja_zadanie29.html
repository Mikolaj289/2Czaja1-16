Zadanie 29 – JavaScript

1. Jak sprawdzić właściwości obiektu?

Można kropką np. auto.marka = "Opel" albo tablicowo auto["rok"] = 1996

2. Odwoływanie się do pól i metod

Robimy to przez kropkę: osoba.imie, osoba.pokaz()

3. Cechy konstruktora

Funkcja tworząca obiekt, może mieć parametry, używa się new, nie zwraca nic

4. Obiekt – przykład

var osoba = {
  imie: "Mikołaj",
  nazwisko: "Czaja",
  zawod: "uczeń",
  pokaz: function(){ console.log(this.imie + " " + this.nazwisko) }
}
osoba.pokaz();

5. Znaczenie this

Odwołuje się do aktualnego obiektu w którym jesteśmy

6. Konstruktor w JS

function Klient(imie, nazwisko, zawod){
  this.imie = imie
  this.nazwisko = nazwisko
  this.zawod = zawod
}
var k = new Klient("Mikołaj", "Czaja", "uczeń")

7. Słowo kluczowe new

Tworzy nowy obiekt: var o = new Klient("Anna", "Nowak", "kierowca")

8. Właściwość prototype

Klient.prototype.pokaz = function(){
  console.log(this.imie + " " + this.nazwisko)
}
